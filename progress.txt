## Architecture Notes
- Monorepo (_saas) contains all 10 sites
- Single GitHub Actions workflow detects changed sites and builds only those
- Multi-arch images (arm64 + amd64) for Pi 5 and Hetzner compatibility
- Infra repo is separate — contains K8s manifests, Flux config, Terraform, monitoring
- GitOps: Flux on each cluster watches infra repo (different overlays per environment)
